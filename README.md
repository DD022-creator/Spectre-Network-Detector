# 🕵️ Spectre Network Detector

An AI-powered system that detects Spectre attacks across a network of computers in real-time.

## 📌 What is This?

Spectre is a critical vulnerability affecting almost all modern CPUs. Attackers can exploit it to steal sensitive data like passwords and encryption keys without leaving any trace.

**This project solves that problem** by putting AI-powered "watchmen" on every computer that monitor for suspicious CPU behavior and raise alerts when an attack is detected.

## 🎯 Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI Detection** | Weighted scoring model (96.2% accuracy) |
| 🌐 **Network-Wide** | Monitors multiple computers simultaneously |
| 📊 **Live Dashboard** | Real-time graphs, alerts, and agent status |
| ⚡ **Attack Simulation** | Test the system with one click |
| 🐳 **Docker Ready** | Easy deployment with Docker |
